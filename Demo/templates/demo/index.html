Описание

В данной версии системы установлены только базовые модули которые являются частью "ядра" системы поставляются вместе с системой.

  • Pages - Страницы
  • Comments - Комментарии к любому типу материалов на сайте
  • Core - Ядро системы. Обязательный модуль для работы.
  • Mail - Почтовый компонент. Хранит все отправленные письма. Мониторинг отправленных писем. Почтовые шаблоны.
  • Menu - Управление меню на сайте.
  • Meta - Работа с мета информацией. Позволяет добавлять мета информацию к любому url адресу сайта.
  • Sites - Мультисайтинг.
  • User - Пользователи. Группы пользователей. Управление правами доступа.
  • Redirect - Работа с преренаправлениями (301, 302 редиректы)
  • Files - Работа с файлами (в т.ч. с внешними облачными хранилищами)

Так же доступны 30+ (суммарно) модулей на Github

Система является полностью бесплатным и открытым продуктом. Исходные коды системы доступны на Github
